1. Differentiate between flex row and flex column

    Flex row

  • In this items are displayed in row wise manner
  • The flex direction default property is Row
  • The items start from the start edge of the main axis.

    • Flex column

  • In this, items are displayed in column wise manner
  • In this the flex-box items are ordered the same way as the text direction, along the cross-axis.

2. Explain Inline, Internal and External CSS

    Inline CSS

  • Inline CSS contains the css property inside the body section attached with the element.
  • Inline Css is given the highest priority.
  • This style is used within HTML tag using the Style Attribute.
  • Its also used to quick check the style property in web page.

    • Internal CSS

  • An Internal Css is used to define style for a single HTML page.
  • Internal Css is used inside the Head section, within a Style element.
  • After inline Css, Internal Css is given the highest priority.

    • External CSS

  • It is basically used when one want to make changes to multiple pages.
  • It uses the link tag in every pages , and it should be written inside head section
  • It has the least priority, but mostly in use when to make change in multiple pages.

3. Justify-Content allows you to do what ?

          Justify-Content allows you to align the flexible container's items when the items do not use the whole space through the main-axis(Horizontally)
             Some of the values of Justify-Content is mentioned below
                 1.flex-start
                 2.flex-end
                 3.center
                 4.space around
                 5.space evenly
                 6.space between

4. Differentiate between Absolute And Relative Positioning

    Absolute Positioning

  • It positons an element based on its ancestor positon
  • it changes the layout around them.

    • Relative Positioning

  • It positions an element based on its current positon.
  • it changes without changing the current layout

The grid-template-column property specifies the number of colums in a grid layout.

6. what is the Z-index in Css

  • It is a Css property that allows you to overlap the HTML elements.
  • Element with the higher index is placed on top of the lower index.
  • Z-index only works on the positioned elements like Absolute or Relative or Fixed.

7. What is the difference between margin and padding ?

    Padding

  • Padding is the space between an element's border and the element's content.
  • It can be defined as the property which controls space inside an element.

    • Margin

  • margin is the space around an element's border.
  • it can be defined as the property that controls the space outside the element.
